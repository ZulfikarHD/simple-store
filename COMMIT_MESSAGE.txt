feat(admin): implementasi manajemen produk dan kategori

Implementasi lengkap Sprint 3 untuk admin panel F&B Web App dengan fitur manajemen produk dan kategori, yaitu: CRUD operations, filtering, search, pagination, image upload, dan constraint validation.

## Backend Changes

### Services
- Add ProductService dengan methods untuk CRUD operations, yaitu:
  - getFilteredProducts() dengan pagination, search, dan filter
  - createProduct() dengan image upload handling
  - updateProduct() dengan partial image update
  - deleteProduct() dengan constraint check untuk prevent delete produk di order aktif
- Add CategoryService dengan methods untuk CRUD operations, antara lain:
  - getAllCategories() dengan product count dan sort order
  - createCategory() dengan auto sort_order generation
  - updateCategory() dengan image handling
  - deleteCategory() dengan constraint check untuk prevent delete kategori yang memiliki produk

### Controllers
- Add ProductController resource controller dengan dependency injection ProductService
- Add CategoryController resource controller dengan dependency injection CategoryService
- Implement Inertia response untuk Vue pages
- Add flash message support untuk success/error feedback

### Form Requests
- Add StoreProductRequest untuk validasi create produk dengan rules, antara lain:
  - Required: name, price, stock, category_id
  - Optional: description, image, is_active, is_featured
  - Image validation: mimes jpeg,png,jpg,webp, max 2MB
- Add UpdateProductRequest untuk validasi update produk
- Add StoreCategoryRequest untuk validasi create kategori dengan unique name constraint
- Add UpdateCategoryRequest untuk validasi update dengan ignore self pada unique rule
- Custom error messages dalam Bahasa Indonesia untuk semua validations

## Frontend Changes

### Pages
- Add Products/Index.vue dengan fitur, yaitu:
  - Data table dengan responsive layout
  - Search bar dengan debounce 300ms menggunakan useDebounceFn
  - Filter dropdown untuk kategori dan status
  - Pagination dengan prev/next navigation
  - Delete confirmation dialog
  - Flash message display
- Add Products/Create.vue untuk form tambah produk dengan:
  - 2-column layout (main content + sidebar)
  - Image upload dengan preview dan remove button
  - Form validation dengan error display per field
  - Loading state untuk submit button
- Add Products/Edit.vue untuk form edit produk dengan:
  - Pre-filled data dari existing product
  - Existing image display dengan option untuk replace
  - Same validation sebagai create form
- Add Categories/Index.vue dengan:
  - Modal form untuk inline create/edit
  - Product count display per kategori
  - Sort order display
  - Delete button disabled jika kategori memiliki produk
  - Constraint validation message
- Add Categories/Create.vue dan Edit.vue untuk dedicated pages (alternatif modal)

### Components & Utilities

  - Produk (Package icon)
  - Kategori (FolderTree icon)

- Add flash message support di HandleInertiaRequests middleware

### Routing
